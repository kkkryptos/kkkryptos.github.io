蓝桥杯_LED

2022-05-03 竞赛 嵌入式 457 字
stm32 蓝桥杯

一、原理图

芯片：SN74HC573ADWR锁存器

PD2：LE 锁存（使能）引脚

点亮方法：PC8-PC15接低电平

二、CubeMX工程配置

1、Pinout&Configuration_System Core

1.SYS

Debug：Serial Wire（串行）

PA13 SYS_JTCK-SWCLK

PA14 SYS_JTMS-SWDID

2.RCC(时钟配置)

High Speed Clock(HSE)：Crystal/Ceramic Resonator（晶振）

PF0-O RCC_OSC_IN

PF1-O RCC_OSC_OUT

3、GPIO
PD2 GPIO_OUTPUT
PC8 GPIO_OUTPUT
PC9 GPIO_OUTPUT
PC10 GPIO_OUTPUT
PC11 GPIO_OUTPUT
PC12 GPIO_OUTPUT
PC13 GPIO_OUTPUT
PC14 GPIO_OUTPUT
PC15 GPIO_OUTPUT

PD8-PD15:

GPIO output level：High（默认状态）——LED默认关闭

GPIO mode：Output Push Pull（推挽输出：可接高低电平）

Maximum Output speed：High（切换电平状态速度）

User Label：LED1-LED8（标签）

PD2：

GPIO output level：High（默认状态）——LED使能打开

GPIO mode：Output Push Pull（推挽输出：可接高低电平）

Maximum Output speed：High（切换电平状态速度）

User Label：LED1_EN（标签）

2、Clock Configuration（时钟源配置）

配置HSE（外部晶振）

Imput frequency：24——HSE——PLLCLK倍频——HCLK：170（回车）

3、Project Manager

1.Project

Project Name(STM32G431RBT6_LED_DEMO)

Project Location（\CubeMX_Keil)

Toolchain/IDE：MDK-ARM

Use Default Firmware Location(取消勾选)——Browse：V1.3.0

2.Code Generator

勾选Generate peripheral initialization as a pair of ‘.c/.h’ fire per peripheral

4、GENERATE CODE

三、代码编写

1、点亮

```
HAL_GPIO_WritePin(LED_EN_GPIO_Port,LED_EN_Pin,GPIO_PIN_SET);//GPIO_PIN_SET高电平，使能

while（1）
{
	HAL_GPIO_WritePin(LED1_GPIO_Port,LED1_Pin,GPIO_PIN_RESET);//GPIO_PIN_RESET低电平,灯亮
	HAL_Delay(200);//SysTick
	HAL_GPIO_WritePin(LED1_GPIO_Port,LED1_Pin,GPIO_PIN_SET);//GPIO_PIN_SET高电平,灯灭
	HAL_Delay(200);
}

使用TogglePin简化⬇️

```
HAL_GPIO_WritePin(LED_EN_GPIO_Port,LED_EN_Pin,GPIO_PIN_SET);//GPIO_PIN_SET高电平，使能

while(1)
{
	HAL_GPIO_TogglePin(LED1_GPIO_Port,LED1_Pin);//HAL_GPIO_TogglePin()翻转
    
	HAL_Delay(200);//SysTick
}

直接对寄存器操作⬇️

```
uint16_t temp=0xFF00;//PC端口有十六位，PC8-15为高八位
HAL_GPIO_WritePin(LED_EN_GPIO_Port,LED_EN_Pin,GPIO_PIN_SET);//GPIO_PIN_SET高电平，使能

while(1)
{
	temp=~temp;//取反
    GPIOC->ODR=temp;//ODR为输出数据寄存器，对其赋值
	HAL_Delay(200);//SysTick
}

2、流水灯

```
uint16_t temp=0xFF00;//PC端口有十六位，PC8-15为高八位
uint16_t i=0;
HAL_GPIO_WritePin(LED_EN_GPIO_Port,LED_EN_Pin,GPIO_PIN_SET);//GPIO_PIN_SET高电平，使能

while(1)	
{
	for(i=0;i<8;i++){
	temp=(0xff00&(0xfeff<<i));//左移i位，取与保证高八位为0
	GPIOC->ODR=temp;
	HAL_Delay(200);}
	i=0;
}
